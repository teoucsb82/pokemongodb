# PokemonGoDB
Quick Pokemon Go Database featuring stats on all available Pokemon, Moves, Types, etc.

## Quick Installation
To install on your system, run:

```
gem install 'pokemongodb
```

Or, to layer into a rails application, add this to your gemfile

```
gem 'pokemongodb'
```
